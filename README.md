# 🌾 Farmo – Farmers to Consumer Marketplace
## web-designing-25bcd10101-25bcd10114

## 📌 Project Overview

Farmo is a simple web-based platform that connects farmers directly with consumers.
The main goal of this project is to eliminate middlemen and provide fair pricing for farmers while ensuring fresh and affordable products for consumers.

---

## 🚀 Features

* Clean and simple user interface
* Navigation bar for easy access
* Home page with hero image
* Separate sections for Farmers and Consumers
* Product cards with images and prices
* “View More” dropdown feature to show additional products
* Contact page for user interaction

---

## 🛠️ Technologies Used

* HTML
* CSS
* Basic UI Design
